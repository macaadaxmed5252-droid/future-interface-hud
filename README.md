# 🌐 Future Interface: Cyberpunk Identity HUD & Dashboard

Future Interface waa bog loogu talagalay maamulka aqoonsiga mustaqbalka (Cyberpunk Dashboard). Wuxuu isku dhex dhabbaa nashqadaynta casriga ah ee **Tailwind CSS** (iyadoo la adeegsanayo Custom Tailwind Configurations), animations firfircoon, iyo dynamic features u ogolaanaya isticmaalaha inuu si toos ah u beddelo sawirka iyo magaca shaashadda.

---

## ✨ Awoodaha Boggan (Features)

* 🔮 **Futuristic Neon Layout:** Interface ku dhisma midabada cyan, indigo, iyo slate, oo leh saameynta muraayadda la dhalaaliyay (*Glassmorphic panels*).
* ⌨️ **Automatic Typing Terminal:** Terminal dhinaca bidix ku yaal oo si otomaatig ah u qoraya koodhadh dynamic ah (sida initialization-ka nidaamka, JIT compilation, iwm).
* 🆔 **Live Identity Override:** Sanduuq qoraal ah oo kuu ogolaanaya inaad magaca terminal-ka ee dhexe u beddesho magac kasta oo aad adigu leedahay si toos ah (*Real-time DOM Manipulation*).
* 🖼️ **Bio-Data Image Upload:** Waxaad toos u soo upload-gareyn kartaa sawirkaaga adoo isticmaalaya **JavaScript FileReader API**, sawirkaas oo isla markiiba noqonaya kan Profile-ka dashboard-ka.
* 🌀 **Dual Rotating Rings:** Labo gooboood oo u kala wareegaya labo dhinac oo iska soo horjeeda si ay u siiyaan badhtamaha bogga muuqaal holographic ah.

---

## 🛠️ Tiknoolajiyada la isticmaalay (Tech Stack)

* **HTML5 & Vanilla JavaScript** - Laf-dhabarta bogga iyo maamulka Logic-ka sida akhrinta faylasha iyo qoraalka iskii u qoraya.
* **Tailwind CSS (Advanced Config)** - Waxaa lagu dhex qoray custom keyframes iyo animations (`animate-scan`, `animate-spin-slow`, `animate-pulse-fast`).
* **FileReader API** - Si sawirka uu qofku soo geliyo loogu beddelo Base64 url oo loogu soo bandhigo shaashadda si dhakhso ah.
* **Google Fonts** - Font-ka mashiinnada ee `Fira Code` iyo kan mustaqbalka ee `Orbitron`.
